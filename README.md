# 📌 Overview

**Kisan Bazar** is a farmer-to-consumer marketplace designed to empower farmers by eliminating middlemen and providing direct access to buyers.

The platform enables farmers to list fresh produce and manage orders, while buyers can explore traditional Indian crops and place orders easily.

The system focuses on **simplicity, accessibility, and transparency** in agricultural trade.

---

# ✨ Features

### 👨‍🌾 Farmer Features

* Farmer authentication and dashboard
* Add and manage product listings
* Track incoming orders
* View sales performance
* Generate reports

### 🛒 Buyer Features

* Browse marketplace products
* Filter products by category and price
* Place orders directly from farmers
* Track order status

### 🌐 Platform Features

* Responsive design for mobile, tablet, and desktop
* Traditional Indian crops focus
* Simple and lightweight backend
* Interactive dashboard with charts

# 🚀 Getting Started

## Prerequisites

Make sure you have installed:

* Node.js
* npm

---

## 1️⃣ Clone the Repository

```
git clone <repository-url>
cd farmers-marketplace
```

---

## 2️⃣ Install Backend Dependencies

```
cd backend
npm install
```

---

## 3️⃣ Start the Server

```
npm start
```

---

## 4️⃣ Open in Browser

```
http://localhost:3000
```

---

# 🧭 Usage Guide

## For Farmers (किसान)

1. Register as a **Farmer**
2. Login to access the **Farmer Dashboard**
3. Add products and manage listings
4. Track incoming orders
5. Monitor sales analytics

---

## For Buyers (ग्राहक)

1. Register as a **Buyer**
2. Browse products in the **Marketplace**
3. Filter by category and price
4. Add items to cart and checkout
5. Track order status

# ⚠️ Known Limitations (Hackathon Prototype)

This project was built during a hackathon and may have some limitations:

* Uses JSON-based storage instead of a full database
* Limited authentication security
* No online payment integration yet
* No real-time inventory updates

Future versions will address these limitations.
